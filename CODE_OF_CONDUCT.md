# Code of Conduct

Mnemos follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be respectful. Disagree on technical merit, not people.
- Assume good intent; bring up concerns directly and privately first.
- No harassment, discrimination, or personal attacks — in issues, PRs,
  discussions, or any other project space.
- Maintainers reserve the right to remove comments, close PRs, or ban
  contributors who violate the code of conduct.

## Reporting

If you witness or experience unacceptable behaviour, contact the maintainer
privately (email on the GitHub profile) or via
[GitHub Security Advisories](https://github.com/jamyouss/mnemos/security/advisories/new)
for sensitive matters.
